### RANK OVER(오라클) 113번
```
RANK()
: 중복 순위가 되면 중복으로 출력 후 중복된 횟수만큼 다음 등수가 없어짐(1등이 2명이면, 1등 출력 후 2등은 출력이 안됨.)

    ex) 결과값 : 1등, 1등, 3등, 4등 ...

DENSE_RANK()
: 중복 순위가 존재해도 다음 순위 값 표시함

    ex) 결과값 : 1등, 1등, 2등, 3등 ...

*RANK 함수 사용시 항목별로 순위를 매기려면 partition by와 같이 사용한다.
```
