```
윈도우 함수 개념
    1. 윈도우 함수는 행과 행 간의 관계를 정의하기 위해서 제공되는 함수
        (행 순서 관련 함수)

    2. 윈도우 함수를 사용해서 순위, 합계, 평균, 행 위치 등을 조작할 수 있다.
        (순위함수, 집계함수)

    3. 윈도우 함수는 GROUP BY 구문과 병행하여 사용할 수 없다.
        (그룹함수)

    4. 윈도우 함수로 인해 결과 건수가 줄어들지는 않는다.

    5. WINDOW 함수의 PARTITION 구문과 GROUP BY 구문은 둘 다 파티션을 분할한다는 의미에서는 유사하다.

    6. sum, max, min 등과 같은 집계 윈도우 함수를 사용할 때 윈도우 절과 함께 사용하면 집계 대상이 되는 레코드 범위를 지정할 수 있다.

    7. 윈도우 함수에는 "OVER" 문구가 키워드로 필수 포함된다.
```

```
1. 윈도우 함수 종류
    1-1) 순위함수
        1-1-1) RANK -- 순위를 계산합니다. 동일한 순위는 동일한 값이 부여됩니다. ex)1,1,3,4,4,6..
        1-1-2) DENSE_RANK -- 동일한 순위를 하나의 건수로 계산합니다. ex)1,1,2,3,3,4..
        1-1-3) ROW_NUMBER -- 동일한 순위에 대해 고유의 순위를 부여합니다. ex)1,2,3,4,5..
    1-2) 행 순서 관련 함수
        1-2-1) FIRST_VALUE('컬럼명') = 파티션에서 가장 처음에 나오는 값을 구한다(MIN 함수를 사용해 같은 결과)
        1-2-2) LAST_VALUE('컬럼명') = 파티션에서 가장 마지막에 나오는 값을 구한다(MAX 함수를 사용해 같은 결과)
        1-2-3)  LAG('컬럼명') = 이전 행(값)을 가져옵니다.
        1-2-4) LEAD('컬럼명',지정값,null을 처리하는값) = 지정된 행(값)을 가져옵니다. DEFAULT값은 1입니다.
                ex)LEAD(sal,1,0) = 결과 출력에서 지정값이 1일 경우 다음행의 데이터를 가져옴, null을 처리하는값이 0일 경우 null을 0으로 출력
    1-5) 집계함수
        1-5-1) SUM(), AVG(), MAX(), MIN(), COUNT()
    1-5) 비율 관련 함수
```
